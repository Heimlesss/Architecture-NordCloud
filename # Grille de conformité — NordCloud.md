# Grille de conformité — NordCloud
 
| Référentiel | Application dans le projet |
|---|---|
| RGPD | Les données personnelles sont protégées. La base de données n’est pas accessible publiquement. Les accès sont limités aux composants nécessaires. |
| ISO 27017 | Les responsabilités cloud sont clarifiées entre le client et le fournisseur. Les Security Groups, IAM et le chiffrement sont documentés. |
| SecNumCloud | L’architecture suit une logique de sécurité renforcée : isolation des tiers, chiffrement, contrôle des accès et limitation des privilèges. |

## Pourquoi choisir OVH ?
 
Choisir OVH comme fournisseur de cloud est une solution adaptée aux besoins car il est déjà certifié SecNumCloud, c'est un acteur français du Cloud qui sépare ses datacenter afin que les règlementations du RGPD soit respectées. :)
